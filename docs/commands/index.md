# Commands Overview

NoJS CLI exposes five commands. Each has a single-letter alias for faster typing.

| Command | Alias | Purpose |
|---------|-------|---------|
| [`nojs init`](init.md) | `i` | Scaffold a new No.JS project via interactive wizard |
| [`nojs prebuild`](prebuild.md) | `b` | Run the build-time optimization plugin pipeline |
| [`nojs dev`](dev.md) | `d` | Start a local dev server with live reload |
| [`nojs validate`](validate.md) | `v` | Validate No.JS templates for common mistakes |
| [`nojs plugin`](plugin.md) | `p` | Search, install, update, and remove plugins |

## Global Options

```
nojs --help       Show usage and command list
nojs --version    Print the installed CLI version
```
